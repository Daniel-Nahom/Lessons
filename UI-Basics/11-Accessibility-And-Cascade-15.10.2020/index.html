Accessibility and Cascade

1. Accessibility

A great deal of web content can be made accessible just by making sure the correct Hypertext Markup Language elements are used for the correct purpose at all times.

Semantic elements are elements with a meaning: for example, if you need a button, use the button tag, not a div.

Semantic elements (h1, h2, ..., p, table, button, ...) clearly define their content.

1.1 Headings

Search engines use the headings to index the structure and content of your web pages.

1.2 Alternative text in images

The alt attribute provides an alternate text for an image, if the user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader).

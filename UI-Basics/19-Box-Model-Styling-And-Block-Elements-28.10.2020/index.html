Block, Inline-Block and Inline elements

Using display: block (divs are block by default)

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).

Using display: inline-block

Compared to display: block, the major difference is that display: inline-block does not add a line-break after the element, so the element can sit next to other elements.

Using display: inline

An inline element does not start on a new line and it only takes up as much width as necessary.

span elements are inline by default.

The properties width, height, margin-top, margin-bottom and float have no effect on inline elements.

As mentioned in the MDN docs, float sometimes modifies the computed value of display.
For example, if you apply a float rule to an inline element, it becomes a block element.
